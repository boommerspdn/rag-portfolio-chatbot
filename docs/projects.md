### Full-Stack Restaurant Management System (AI OCR)
A full-featured restaurant operations app. The flagship feature is an **AI-powered OCR pipeline** using Gemini API / Vertex AI to extract structured JSON data from handwritten order images, automating order entry and reducing manual data input by 70%.

Additional features:
- Real-time inventory / stock management.
- Daily revenue summary dashboard.
- Real-time order status tracking per table or order, with structured approval flows for order state transitions.
- Engineered with Next.js and TanStack Table to handle 100+ concurrent orders with high-performance sorting and data filtering.
- PostgreSQL database with Prisma ORM, containerized with Docker.
- Delivered as a Next.js PWA for fast load times and app-like usage across devices.

GitHub: https://github.com/boommerspdn/kruakoonpim-manager
Live: https://kruakoonpim-manager.vercel.app
